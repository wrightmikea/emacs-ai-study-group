# aider.el Demo

This project demonstrates [aider.el](https://github.com/tninja/aider.el), an Emacs interface for Aider, an AI pair programming tool.

## Overview

aider.el provides an Emacs interface to [Aider](https://aider.chat/), allowing you to:
- Chat with AI about your code directly from Emacs
- Make AI-powered code changes
- Leverage Aider's git integration
- Use various LLM providers (OpenAI, Anthropic, local models)

## Setup

1. Install Aider: `pip install aider-chat`
2. Install aider.el (configuration instructions coming soon)
3. Configure your API keys for your preferred LLM provider
4. Start Aider with `M-x aider`
