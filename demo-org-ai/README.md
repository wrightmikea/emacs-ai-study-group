# org-ai Demo

This project demonstrates [org-ai](https://github.com/rksm/org-ai), an Emacs package for using AI in org-mode.

## Overview

org-ai integrates AI capabilities directly into org-mode, allowing you to:
- Chat with AI models in org documents
- Generate text inline
- Refactor and improve code
- Create images with DALL-E
- Use speech input/output

## Setup

1. Install org-ai from MELPA
2. Configure your OpenAI API key
3. Use `#+begin_ai` blocks in your org files
