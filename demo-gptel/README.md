# gptel Demo

This project demonstrates [gptel](https://github.com/karthink/gptel), a simple LLM client for Emacs.

## Overview

gptel is a simple, no-frills LLM client for Emacs that supports:
- ChatGPT (OpenAI API)
- Claude (Anthropic API)
- Gemini (Google AI API)
- Local models via Ollama, Llama.cpp, and others

## Setup

1. Install gptel from MELPA
2. Configure your API keys
3. Start chatting with `M-x gptel`
