# Emacs AI Study Group

This repository contains demonstration projects and examples for the Emacs AI Study Group.

## Projects

### [demo-aider-el](./demo-aider-el/)

Demonstrations and examples of using aider.el, an Emacs interface for Aider AI pair programming.

### [demo-elisp-dev-mcp](./demo-elisp-dev-mcp/)

Demonstrations and examples of using elisp-dev-mcp, a Model Context Protocol server for Emacs Lisp development.

### [demo-ellama](./demo-ellama/)

Demonstrations and examples of using Ellama, an Emacs tool for interacting with LLMs using Ollama.

### [demo-gptel](./demo-gptel/)

Demonstrations and examples of using gptel, a simple LLM client for Emacs that supports ChatGPT, Claude, Gemini, and local models.

### [demo-org-ai](./demo-org-ai/)

Demonstrations and examples of using org-ai, which integrates AI capabilities directly into org-mode.

## Getting Started

Each project directory contains its own README with setup instructions and examples.
