# Ellama Demo

This project demonstrates [Ellama](https://github.com/s-kostyaev/ellama), an Emacs tool for interacting with LLMs using Ollama.

## Overview

Ellama is an Emacs tool that provides:
- Interactive LLM sessions directly in Emacs
- Code generation and refactoring
- Text translation and summarization
- Context-aware assistance using local or remote models via Ollama
- Integration with various LLM providers

## Setup

1. Install Ollama: [https://ollama.ai/](https://ollama.ai/)
2. Pull a model: `ollama pull llama2` (or your preferred model)
3. Install Ellama from MELPA
4. Configure Ellama to use your preferred model
5. Start using Ellama with `M-x ellama-chat`
