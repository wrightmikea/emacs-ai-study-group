# elisp-dev-mcp Demo

This project demonstrates [elisp-dev-mcp](https://github.com/ppbb/elisp-dev-mcp), a Model Context Protocol (MCP) server for Emacs Lisp development.

## Overview

elisp-dev-mcp provides an MCP server that enables AI assistants to:
- Access and manipulate Emacs Lisp code
- Query Emacs documentation
- Interact with Emacs buffers and processes
- Assist with Elisp development workflows

## Setup

1. Install the elisp-dev-mcp server
2. Configure your MCP client (e.g., Claude Desktop) to connect to the server
3. Start using AI assistance for your Elisp development
